Detailed Description

Implementation of DIMACS CNF format writer.

This file implements the DimacsWriter class which outputs CNF formulas in DIMACS format, the standard input format for SAT solvers.

DIMACS CNF Format Specification:

  • Comment lines start with 'c'
  • Problem line: "p cnf <num_vars> <num_clauses>"
  • Clauses: Space-separated literals terminated by '0'
  • Literals: Positive integers (variable) or negative integers (negated variable)

Example:

c This is a comment
c 1 FeatureA
c 2 FeatureB
p cnf 2 3
1 2 0
-1 0
2 0
