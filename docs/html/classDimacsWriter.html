Detailed Description

Writes CNF models to DIMACS format.

DimacsWriter converts a CNFModel to the standard DIMACS CNF format, which is widely used by SAT solvers. The DIMACS format consists of:

Header Line:

  • Format: p cnf <num_variables> <num_clauses>
  • Example: p cnf 5 10 (5 variables, 10 clauses)

Comment Lines (optional):

  • Start with 'c' character
  • Used for variable mappings and auxiliary variable descriptions
  • Example: c 1 GPS, c 2 Display

Clause Lines:

  • Each clause on a separate line
  • Literals separated by spaces
  • Terminated with 0
  • Positive integers for variables, negative for negations
  • Example: 1 -2 0 represents (v1 | ~v2)
See also
CNFModel for the input CNF representation
FMToCNF for converting feature models to CNF

Example output:

c Variable mapping
c 1 GPS
c 2 Display
p cnf 2 2
1 0
-1 2 0

void write_to_stream ( std::ostream &  out)

Writes the CNF model to an output stream in DIMACS format.

Parameters
outThe output stream to write to (e.g., std::cout, file stream)

Output format:

  1. Problem line: "p cnf <variables> <clauses>"
  2. Comment lines for feature variables: "c <var_id> <feature_name>"
  3. Comment lines for auxiliary variables: "c <var_id> <aux_name>"
  4. Clauses: "<lit1> <lit2> ... 0" (one per line)
